## 功能描述

HC05蓝牙模块是一种主从一体的蓝牙串口模块。当两个蓝牙设备成功配对并连接后，用户可以忽略蓝牙内部的通信协议，直接将蓝牙模块当做串口使用。一旦连接建立，两个设备将共享同一个通信通道（即串口），一个设备发送的数据可以直接被另一个设备接收。
